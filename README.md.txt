* Search NSE-listed companies
* Interactive historical charts
* Save favorite stocks in a watchlist


## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sahilboken01/stocksense.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
