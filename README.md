StockSense

StockSense is a full-stack stock analytics dashboard that allows users to track NSE stocks, search companies, view live market data, analyze historical price trends, and manage a personalized watchlist.

🚀 Live Demo

https://stocksense-swart.vercel.app/

✨ Features
Live NSE stock tracking
Stock search functionality
Historical price charts
Personal watchlist with local storage
Responsive modern UI
FastAPI REST API backend
Real-time stock market data
Full-stack deployment with Vercel and Render

⚙️ Installation
Clone Repository
git clone https://github.com/sahilboken01/stocksense.git
Frontend Setup
cd frontend
npm install
npm run dev
Backend Setup
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
